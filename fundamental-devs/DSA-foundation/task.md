# Mastering the Basics of Data Structures and Algorithms (DSA)

Mastering DSA requires solving problems that cover fundamental logic, data structures, and core algorithms. Here's a curated list of 30 basic DSA problems categorized by topic to help build a solid foundation.

## Logic, Loops, & Basic Math
These problems help practice basic programming syntax and logic.

1. **Find Factorial**: Calculate *n!* using iterative and recursive methods.  
2. **Check Prime**: Determine if a number is prime.  
3. **Fibonacci Series**: Generate the first *n* numbers in the Fibonacci sequence.  
4. **Palindrome Number**: Check if a number reads the same backward as forward.  
5. **Armstrong Number**: Check if the sum of the cubes of a number's digits equals the number itself.  
   *(Problems: 3, 4, 5, 6, 7)*

## Arrays & Hashing
Arrays are common starting points for data structure problems.

6. **Two Sum**: Find two numbers in an array that add up to a specific target.  
7. **Find Max/Min**: Locate the largest and smallest elements in an array.  
8. **Reverse an Array**: Invert the order of elements in place.  
9. **Majority Element**: Find the element that appears more than *n/2* times.  
10. **Contains Duplicate**: Use a hash set to check for repeating values.  
11. **Move Zeroes**: Shift all zeros to the end of an array without changing the order of other elements.  
   *(Problems: 8, 9, 10)*

## Strings
Mastering string manipulation is essential for technical interviews.

12. **Valid Palindrome**: Check if a string is a palindrome, ignoring case and non-alphanumeric characters.  
13. **Valid Anagram**: Check if two strings contain the same characters in different orders.  
14. **Reverse a String**: Invert the characters of a string.  
15. **First Unique Character**: Find the index of the first character that does not repeat.  
16. **Longest Common Prefix**: Find the longest prefix shared by a set of strings.  
   *(Problems: 11, 12, 13)*

## Linked Lists
Linked lists introduce memory pointers and non-contiguous storage.

17. **Reverse a Linked List**: Change the direction of all pointers in the list.  
18. **Detect Cycle**: Use the "Fast and Slow Pointer" technique to find cycles.  
19. **Merge Two Sorted Lists**: Combine two sorted lists into one.  
20. **Middle of the Linked List**: Find the center node in one pass.  
   *(Problems: 14, 15, 16)*

## Stacks & Queues
Linear structures following specific order principles (LIFO/FIFO).

21. **Valid Parentheses**: Use a stack to check if brackets are balanced.  
22. **Implement Queue using Stacks**: Mimic FIFO behavior using two LIFO stacks.  
23. **Min Stack**: Design a stack that retrieves the minimum element in *O(1)* time.  
   *(Problems: 17, 18, 19)*

## Searching & Sorting
Algorithms for organizing and finding data efficiently.

24. **Binary Search**: Find an element in a sorted array in *O(log n)* time.  
25. **Bubble Sort**: Practice basic element swapping for sorting.  
26. **Merge Sort**: Learn the "Divide and Conquer" approach for *O(n log n)* sorting.  
   *(Problems: 20, 21, 22)*

## Trees & Recursion
Trees are fundamental for hierarchical data.

27. **Maximum Depth of Binary Tree**: Calculate the height of a tree.  
28. **Inorder Traversal**: Visit nodes in order (Left, Root, Right).  
29. **Symmetric Tree**: Check if a tree is a mirror of itself.  
30. **Leaf-Similar Trees**: Determine if two trees have the same sequence of leaves.  
   *(Problems: 23, 24, 25)*
